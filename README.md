# GRIZIP  🇬🇧/🇺🇸



## Description
GRIZIP is an implementation of the Zip logic game. In this game, players must connect numbers in ascending order by tracing a continuous path on a grid. Each cell can only be used once, and the difficulty level evolves throughout the week (from Monday to Sunday).

**Inspired by Zipgame from LinkedIn.**

## 🎮 Game Rules
- Start with number 1 and connect all numbers in ascending order
- The path must pass through every cell of the grid exactly once
- Movements can only be made horizontally or vertically (not diagonally)
- Each level must be completed by connecting all numbers and passing through all cells

## 🛠️ Technologies Used
- React 18
- TypeScript
- SWC (compiler)
- Vite (build tool)
- CSS Modules

## ✨ Features

### Implemented

- ✅ Intuitive and responsive user interface

# GRIZIP  🇫🇷

## Description
GRIZIP est une implémentation du jeu de logique Zip. Dans ce jeu, les joueurs doivent relier des nombres dans l'ordre croissant en traçant un chemin continu sur une grille. Chaque cellule ne peut être utilisée qu'une seule fois, et le niveau de difficulté évolue au fil de la semaine (du lundi au dimanche).

**Inspiré par Zipgame de LinkedIn.**

## 🎮 Règles du Jeu
- Commencez par le nombre 1 et connectez tous les nombres dans l'ordre croissant
- Le chemin doit passer par toutes les cases de la grille exactement une fois
- Les déplacements ne peuvent se faire qu'horizontalement ou verticalement (pas en diagonale)
- Chaque niveau doit être complété en reliant tous les nombres et en passant par toutes les cases
